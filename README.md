yii2-curl for framework yii2
=========

Curl yii2 main code https://github.com/shuber/curl

<code>
 $curl = new curl\Curl();
</code> 
 Installation
=========
The preferred way to install this extension is through composer.

Either run
<code>
php composer.phar require --prefer-dist xr0m3oz/yii2-curl "*"
</code>
